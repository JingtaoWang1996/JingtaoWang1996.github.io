## 使用Git

### Git必要的配置

* 顶部打开一个新的terminal，运行：git --version

  * 如果输出版本表示已经识别到git
  * 如果没有输出版本，则在VScode的左下角点击⚙️->setting-> 搜索“git.path"
    * 在打开的setttings.json文件中填写git.path：`  "git.path": "C:\\Program Files\\Git\\cmd\\git.exe"`

* **配置Git用户**，在VSCode终端执行一下命令进行配置：

  ```
  git config --global user.name "JingtaoWang"
  git config --global user.email "wjt397618242@163.com"
  ```

  * 查看配置情况：`git config --global --list`

* **开启VSCode的git功能**

  * VSCode左侧找到**Source Control** 

    * 或者快捷键：Ctrl+Shift+G

    * 如果没有，去设置里确认：`Settings → 搜索 Git: Enabled` 确保勾选

  * 配置完git之后点击左侧诶对Source control会出现 Changes、Graph等

### Git 提交-Commit

* 打开自动保存：File -->AutoSave
* 代码修改后会在左侧Source control的change出现
* 如果要提交，输入message后commit & push
* 如果某一个文件的修改要撤回，直接这个文件右侧的`Discard Changes`后即可恢复此文件全部改动



### Git 拉取代码-Pull

* 终端直接执行：`git pull`

* 指定分支拉取: `git pull origin main`
* master分支拉取: `git pull origin master`
* 开发分支拉取: `git pull origin dev`

### 遇到的问题

* 问题1：git版本无法识别

  * git --version  执行报错：git : 无法将“git”项识别为 cmdlet、函数、脚本文件或可运行程序的名称。请检查名称的拼写，如果包括路径，请确保路径正确，然后再试 一次。

    * 原因：Git 已经安装（你说已安装），但没有加入系统 PATH，VS Code / PowerShell 找不到它。

    * 解决方案：手动添加git到环境变量

      * win+R -->sysdm.cpl-->高级-->环境变量

      * 在系统变量或者用户变量中找到`Path`,新增两行

        ```
        C:\Program Files\Git\bin
        C:\Program Files\Git\cmd
        ```

      * 保存并关闭所有窗口后重新打开VSCode

      * 再次执行git --version问题解决

## VScode 同时打开并运行多个工程

### 使用工作区Workspace

* 打开第一个工程：`File → Open Folder`
* 添加第二个工程：`File → Add Folder to Workspace`
* 保存工作区：`File → Save Workspace As`

**以之后双击这个workspace就会同时打开两个工程**

# VScode for Vue

## 安装VSCode Extension

* 点击VScode左侧**Extension**
* 安装以下扩展：
  * Vue - Official
  * ESLint
  * Prettier - Code formatter

## 已有 Vue 项目 + pnpm + VS Code 运行

* 打开项目根目录：File --> Open Folder

* 打开VScode终端：顶部Terminal--> New Terminal

  * **确认pnpm已经安装：pnpm -v**【输出pnpm的版本号】
    * 如果提示没有pnpm，执行命令安装：npm install -g pnpm
  * 安装项目依赖【项目根目录下执行】：pnpm install 

* 打开`package.json`,搜索其中的`scripts`, 确认项目启动方式

  以leo_vue项目为例，找到的内容如下

  ```
    "scripts": {
      "serve": "vue-cli-service serve",
      "build": "vue-cli-service build",
      "lint": "vue-cli-service lint"
    },
  ```

  * 如果scripts下面的key包含了：“dev”：
    * terminal中执行命令运行项目：`pnpm dev`
  * 如果scripts下面的key包含`serve`[类似上面的示例]：
    * Terminal执行命令运行项目：`pnpm serve`

## 创建Vue项目

* 确认Node.js和npm可用

```
node -v
npm -v
```

### npm创建Vue项目

* 打开VSCode终端：`Terminal-->New Terminal`
* 创建项目，根据提示选择项目配置：`npm create vue@latest`
* 终端cd进入项目：`cd 项目名`
* 安装依赖：`npm install`
* 启动项目:`npm run dev`或 `npm run serve`

### pnpm创建Vue项目

* 安装pnpm：`npm install -g pnpm`
  * 确认安装完成：`pnpm -v`
* 创建项目：`pnpm create vue@latest`
* 终端cd进入项目：`cd 项目名`
* 安装依赖：`npm install`
* 启动项目:`npm run dev`或 `npm run serve`

### 项目创建后用VScode打开

* terminal cd 到项目根目录后执行：`code .`
* File ->Open Folder ->打开根目录
